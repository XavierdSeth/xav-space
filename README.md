# xav·space
xav·space is my personal website where i can show off my projects or write about my experiences.
this site was a good experience learning how to write css and incorporate it into html.
i had set up the js so that i could easily adjust my status, mood, activity, location, time, and music. i was originally going to set it up to link to my spotify, however it was quite complex and i was told that my spotify api key would be public on the site, which i do not want. so instead i decided to just put my faviourite song on instead and make it so i can adjust it whenever. 

### The Website has 4 pages:

Home: the home page contains a paragraph welcoming any visitors to the site, this page also includes a featured project of mine and the latest blog post.

About: the about page contains a paragraph about me and who i am, this page also includes a list of all my interests and a description of this website.

Projects: the project page contains panels of all of the projects i have made with a status badge and a short description.

Blog: the blog page contains any blogs i want to post about anything, it has the date, topic, and main text.

All of these pages have the same side bar which includes what i am currently doing (status, mood, doing, where, time), my current faviourite song. and recent updates on my projects. at the bottom of the sidebar it contains, a link to the website, my personal email, my github link.
